美化生成 Markdown

Hexo 本身并不直接提供美化 Markdown 的功能，但你可以通过以下方式来增强 Markdown 的渲染效果：

  1. 使用主题
    选择一个美观的主题可以显著提升博客的外观。Hexo 有很多主题可供选择，你可以在 Hexo 主题页面 找到它们。

  2. 自定义 CSS
    你可以通过自定义 CSS 文件来调整 Markdown 的渲染样式。在主题的 source/css 目录下创建或修改 CSS 文件，然后在 _config.yml 中引用它。

  3. 使用插件
    有一些插件可以帮助你更好地渲染 Markdown，例如 hexo-renderer-marked 可以增强 Markdown 的渲染效果。

  4. Markdown 扩展
    你可以使用一些 Markdown 扩展语法来增强文章的表现力，例如使用 markdown-it 作为渲染引擎，并启用一些插件。
